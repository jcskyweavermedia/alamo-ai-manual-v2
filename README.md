# Tastly AI - Restaurant Operations Manual

Bilingual (EN/ES) restaurant operations manual with AI assistant.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase (Auth, Database, Edge Functions, Storage)
- OpenAI (gpt-4o-mini, text-embedding-3-small)

## Development

```sh
# Install dependencies
npm i

# Start the development server
npm run dev
```

## Deployment

Edge functions are deployed via Supabase CLI. The frontend can be deployed to any static hosting provider.
