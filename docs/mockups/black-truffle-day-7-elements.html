1 — Section Header
06 — Avoid These

Common Misfires

And exactly how to fix each one

5 — Medium Cards (miss / fix)
Miss Overpromising truffle availability or timing.
Fix Set expectations up front—confirm what's available right now before you sell it.
Miss Offering truffle or creamed spinach after the food's already cooling.
Fix Time the offer before the fire so it lands piping hot in a pre-warmed bowl.
Miss Vague upcharge language: "Do you want truffle?"
Fix Use clear pricing: "We can add truffle for $X—would you like that today?"
Miss Skipping allergy prompts when it matters.
Fix "Any allergies or dairy concerns?" (Creamed Spinach contains dairy).
Miss Sloppy presentation—drips, smeared rims, messy bowls.
Fix Follow foh_plate_specs: clean, wiped rims; neat, intentional finish; hot bowl.
Miss Delayed firing and table gaps.
Fix Coordinate with kitchen early so sides and mains hit together—hot and on pace.
